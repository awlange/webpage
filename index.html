Excited Electrons In DNA

As part of my dissertation, I studied excited electrons in DNA using quantum chemistry simulations. The red-blue blob here is a depiction of how the three dimensional wave (or, orbital) of an electron spreads out between two adenine molecules when it is excited by ultraviolet light. The red and blue indicates the phase of the wave. The adenine dimer is somewhat hidden by the orbital, but the dimer is in the stacking geometry found in DNA and surrounded by water, serving as a model to understand electronic excitation in solvated DNA.

Fragment Molecular Orbital Multi-state Reactive Molecular Dynamics

This is a cute idea I had to combine Fragment Molecular Orbital (FMO) theory with Prof. Greg Voth's Multi-state Empirical Valence Bond (MS-EVB) theory. FMO divides a molecular system into some set of smaller fragments, but this division can be ambiguous in chemically reactive systems. MS-EVB theory deals exactly with that issue by assuming a linear combination of reactive states, like what is shown in the schematic here. Combining MS-EVB and FMO results in a new powerful model that incorporates quantum electronic structure theory into the MS-EVB approach.

Debye-Hückel-like Screening Model

This is an equation I invented to model continuum electrostatics of salty solvents characterized by a Debye screening length and a dielectric constant. I called the equation the "Debye-Hückel-like Screening Model" (DESMO) because it generalizes the classic Debye-Hückel model—a point charge centered in a spherical solute cavity surrounded by solvent continuum—to arbitrary charge densities and solute cavities. DESMO reduces to the more familiar "Conductor-like Screening Model" (COSMO) in the limit of zero salt concentration. DESMO is a simple, fast approximation to much more complicated models.

Genetic Algorithm Load Balancer

The implicit solvent calculation here is based on a parallel fast multipole method, which runs faster when atoms belonging to a given node are spatially close to each other and also far away from atoms belonging to other nodes. The fitness function for the genetic algorithm reflects that, and it also tends to ensure that no nodes have far more/less work than other nodes.

The molecule depicted is a single stranded DNA polyadenine, and each sphere represents an atom. Spheres are colored by which node is responsible for its computation. (There are 4 nodes in this example.) The run starts with a randomized distribution of the atoms across the 4 nodes, which is essentially the worst load balance. Interestingly, the genetic algorithm finds a solution that is very close to the ideal solution for this system.

Massively Parallel Computing

My primary focus as a postdoc was improve the scalability of a code on (at the time) the 4th fastest supercomputer in the world, Mira, the Blue Gene/Q machine at Argonne National Laboratory Leadership Computing Facility.

I managed to do this well for a specialized molecular dynamics code used to simulating proton transport. The original code scaled to only about 64 nodes, but I turned it into one that scales to 24,576 nodes (393,216 cores). To achieve this, I built up a replica exchange external driver to the molecular dynamics code LAMMPS, exposed untapped parallelism in the multi-state proton transport calculations, multithreaded compute-intensive loops, and plenty of other little code optimizations. I also managed to contribute a handful of lines of code to the open-source LAMMPS software during this time.
